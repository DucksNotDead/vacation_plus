# Vacation Plus

Монорепозиторий: frontend (React/Vite) и backend (.NET API).

## Структура

```
frontend/   # клиент (приложение в frontend/app)
backend/    # API (ASP.NET Core)
```

## Frontend

```bash
cd frontend/app
npm i
npm run dev
```

## Backend

```bash
cd backend
dotnet restore
dotnet run
```
